Penetration Testing Lab

Completed a hands-on penetration testing lab using Offensive Security’s Proving Grounds platform. The lab involved exploiting a vulnerable Ubuntu Linux system running Nagios XI. Activities included TCP port scanning with Nmap, vulnerability research using SearchSploit, and exploitation using a Metasploit authenticated remote code execution module. Post-exploitation tasks included system enumeration, privilege validation, file system analysis, and identification of user artifacts. This project demonstrates applied offensive security techniques, structured exploitation methodology, and controlled post-exploitation analysis.
